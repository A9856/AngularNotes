AngularJS क्या है?

        AngularJS एक Open Source JavaScript Framework है जिसका उपयोग Dynamic Web Application और Single Page Application (SPA) बनाने के लिए किया जाता है।
        Developed By: Google
        Released: 2010
        Purpose: Web Page को Dynamic और Interactive बनाना।

        AngularJS की विशेषताएँ--:
            Open Source Framework
            Single Page Application (SPA) बनाता है
            Two Way Data Binding प्रदान करता है
            MVC Architecture को Support करता है
            Directives प्रदान करता है
            Dependency Injection Support करता है
            Testing आसान बनाता है
            AngularJS जोड़ने का तरीका

            HTML Page में AngularJS Library जोड़ने के लिए Script Tag का उपयोग करते हैं।

            

            Directive क्या है?
            Directive AngularJS का विशेष Attribute होता है जो HTML Element को अतिरिक्त Functionality प्रदान करता है।
            AngularJS के सभी Directive सामान्यतः ng- से शुरू होते हैं।

            ng-app Directive--:
            ng-app AngularJS Application की शुरुआत (Starting Point) को Define करता है।
            AngularJS सबसे पहले ng-app खोजता है और वहीं से Application शुरू करता है।

            ng-model Directive--:
            ng-model HTML Input Control को AngularJS Variable से जोड़ता है।यह Two-Way Data Binding का कार्य करता है।

            ng-bind Directive--:
            ng-bind Variable की Value को HTML Element में Display करता है।
           
            Syntax
            
            
            

            {{5+5}}

            ng-model और ng-bind में अंतर

            ng-model--: Input Field को Variable से जोड़ता है।
            

            ng-bind--: Variable की Value Display करता है।
            


            AngularJS Expression--:Double Curly Braces {{ }} के अंदर लिखे गए Statement को Expression कहते हैं।

            Example
            {{10+20}}

            String Example
            {{"Adil"+" Khan"}}
            
            ng-init Directive--:ng-init Variable को Initial Value देने के लिए उपयोग किया जाता है।

            Example
            

            {{color}}
            


            Object in AngularJS--:Object कई Values को Key और Value के रूप में Store करता है।

            Example
            

        
            {{person.name}}
            {{person.city}}
            {{person.age}}
    
            Array in AngularJS--:Array समान प्रकार के Data का Collection होता है।Array का Index हमेशा 0 से शुरू होता है।

            Example
            

            {{student[0]}}
            

            {{student[1]}}
            

            {{student[2]}}
            


            Controller--:Controller AngularJS Application का Logic संभालता है।Controller Data और Business Logic को Manage करता है।

            Example
            app.controller(
            "myCtrl",
            function($scope)
            {
            $scope.name="Adil";
            });

            Scope--:Scope Controller और View के बीच Connection का कार्य करता है।
            $scope AngularJS का एक Special Object है जो Controller और View के बीच Data एवं Functions को Share करने का कार्य करता है।
            Scope Data को Controller से HTML तक पहुँचाता है।

            Example
            $scope.name="Adil";
            HTML
            {{name}}
     
            Data Binding--:Model और View के बीच Data का Automatic Synchronization Data Binding कहलाता है।

            Example
            

            {{name}}
            User जो Input करेगा वही तुरंत Display होगा।
            इसे Two-Way Data Binding कहते हैं।

            Filter--:Filter का उपयोग Data को Format करने के लिए किया जाता है।

            Uppercase Filter
            {{"adil"|uppercase}}

            Lowercase Filter
            {{"ADIL"|lowercase}}

            Currency Filter
            {{5000|currency}}
           
            Service--:Service Reusable Code होता है जिसका उपयोग Application के कई भागों में किया जा सकता है।

            Built-in Services
            $http
            $timeout
            $interval
            $location
           
            Testing--:AngularJS Application में Errors खोजने और Code Verify करने की प्रक्रिया Testing कहलाती है।

            Testing Tools
            Jasmine
            Karma
            Protractor
